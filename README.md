## Description
This project is an educational keylogger developed using Python to understand how keyboard events are captured for cybersecurity awareness.

## Features
- Captures key press and key release events
- Stores data in text and JSON format
- GUI-based control using Tkinter
- Runs only with user interaction

## Technologies Used
- Python
- pynput
- tkinter
- JSON

## How to Run
1. Install dependencies:
   pip install pynput
2. Run the program:
   python keylogger_gui.py

## Ethical Notice
This project is created strictly for educational purposes. Unauthorized keylogging is illegal and unethical.
